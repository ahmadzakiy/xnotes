# xNotes v.1.1.1

An app to write notes on chrome new tab

## GETTING STARTED

```
npm install / yarn install
npm start / yarn start
```

## BUILD CHROME EXTENSION

```
un-commend chrome.storage function at app.js
npm run build
load build folder from chrome browser
```
